## ✨ Storia

> Ho 35 anni. Per 10 anni ho gestito un'azienda. Poi ho scoperto la tecnologia.
>
> Non è stato un cambio di carriera. È stata una rivelazione. La capacità di costruire, dal mio terminale, strumenti che attraversano i confini.
>
> Subvox è nato da questa ossessione: e se potessi prendere qualsiasi video — un discorso politico coreano, un tutorial tedesco, una conferenza in arabo — e renderlo accessibile nella tua lingua madre, in pochi minuti, senza perdere la voce, il tono, l'intenzione?
>
> Subvox significa anche contribuire come comunità per rendere la traduzione video accessibile a tutti. A differenza di **Veed.io**, **Kapwing**, **Descript** o **Opus Clip**, giganti che fanno pagare una fortuna per ciò che dovrebbe essere gratuito, che limitano le tue esportazioni, che ti blindano in abbonamenti. Subvox è aperto, trasparente e costruito dalle persone che lo usano.
>
> Questo è un progetto individuale, costruito commit dopo commit, perché credo che la tecnologia dovrebbe appartenere a tutti. Non solo a chi parla inglese.
>
> Questa pipeline è il cuore tecnico. Il resto — wallet, token, economia — vive altrove. Qui c'è la macchina per tradurre il mondo. Gratuita. Aperta. Tua.
>
> *Nansou*

## 🎯 Funzionalità

| Passaggio | Descrizione |
|---|---|
| **⬇️ Scarica** | Recupera video da X/Twitter, YouTube o URL diretto |
| **🎙️ Trascrivi** | Trascrizione audio tramite Groq (Whisper), 20+ lingue |
| **🌐 Traduci** | Traduzione LLM (DeepSeek / OpenAI) nella lingua target |
| **🎬 Incorpora** | Sovrapposizione sottotitoli nel video (ffmpeg/libass) |
| **☁️ Carica** | Archiviazione locale o S3 del risultato finale |

**Bonus:** Esportazione VTT/SRT, watermark automatico, rilevamento scene, analisi dei contenuti.

---

## 🎬 Panoramica della pipeline

```
1. Incolla un link video
   │  YouTube, X/Twitter, TikTok, Instagram, Facebook...
   │  Il validatore controlla l'accessibilità in 2 secondi
   ▼
2. Download
   │  yt-dlp recupera il video (fino a 4K)
   ▼
3. Trascrizione audio
   │  Groq Whisper converte l'audio in testo
   │  Supporta 20+ lingue
   ▼
4. Traduzione
   │  DeepSeek / OpenAI traduce i sottotitoli
   │  Nella lingua che preferisci
   ▼
5. Incorporamento
   │  ffmpeg + libass incorporano i sottotitoli nel video
   │  Watermark automatico, stile personalizzabile
   ▼
6. Risultato
   │  Video sottotitolato, pronto per essere condiviso
   │  Esportazione SRT/VTT disponibile
```

**Tempo totale:** 2-5 minuti per un video di 3 minuti.
**Nessuna competenza di programmazione richiesta.** Incolla un link, scegli una lingua, ottieni un video tradotto.

## 🚀 Avvio rapido

### Con Docker (consigliato)

```bash
# 1. Clona
git clone https://github.com/Nansouoouu/subvox-pipeline.git
cd subvox-pipeline

# 2. Avvia l'intero stack
export GROQ_API_KEY="your_groq_key"
export DEEPSEEK_API_KEY="your_deepseek_key"
docker compose up -d

# 3. Verifica
curl http://localhost:8000/health
```

### Senza Docker (sviluppo locale)

```bash
# 1. Setup
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 2. Configurazione
cp .env.example .env
# → Modifica DATABASE_URL, REDIS_URL

# 3. Esegui
uvicorn backend.main:app --reload --port 8000
```

## 📡 API

| Route | Metodo | Descrizione |
|---|---|---|
| `/jobs/feed` | GET | Ultime traduzioni pubbliche |
| `/jobs/{id}/status` | GET | Stato del job |
| `/jobs/{id}/subtitles` | GET | Sottotitoli generati |
| `/health` | GET | Health check |

L'autenticazione e la gestione dei token sono gestite da un servizio dedicato (privato).

## 🧩 Stack

| Componente | Tecnologia |
|---|---|
| API | Python 3.14 / FastAPI |
| Coda | Celery + Redis |
| Video | FFmpeg 7 |
| Trascrizione | Groq (Whisper) |
> **Nota:** Groq offre 2 ore di trascrizione gratuita al giorno. Perfetto per lo sviluppo e progetti piccoli.
| Traduzione | DeepSeek / OpenAI |
| DB | PostgreSQL 16 |
