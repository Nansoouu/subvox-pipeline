## ✨ Geschichte

> Ich bin 35. 10 Jahre lang habe ich ein Unternehmen geführt. Dann habe ich die Technologie entdeckt.
>
> Es war keine berufliche Neuorientierung. Es war eine Offenbarung. Die Fähigkeit, von meinem Terminal aus Werkzeuge zu bauen, die Grenzen überwinden.
>
> Subvox entstand aus dieser Besessenheit: Was wäre, wenn man jedes Video nehmen könnte – eine koreanische politische Rede, ein deutsches Tutorial, eine arabische Konferenz – und es in Minuten in seiner Muttersprache zugänglich machen könnte, ohne die Stimme, den Ton, die Absicht zu verlieren?
>
> Subvox geht auch darum, als Gemeinschaft beizutragen, um Videoübersetzung für alle zugänglich zu machen. Anders als **Veed.io**, **Kapwing**, **Descript** oder **Opus Clip**, Giganten, die ein Vermögen für etwas verlangen, das kostenlos sein sollte, die Ihre Exporte einschränken, die Sie in Abonnements sperren. Subvox ist offen, transparent und von denen gebaut, die es nutzen.
>
> Dies ist ein Solo-Projekt, Commit für Commit aufgebaut, weil ich glaube, dass Technologie allen gehören sollte. Nicht nur denen, die Englisch sprechen.
>
> Diese Pipeline ist das technische Herz. Der Rest – Wallets, Tokens, Wirtschaft – lebt woanders. Hier ist die Maschine, um die Welt zu übersetzen. Kostenlos. Offen. Deins.
>
> *Nansou*

## 🎯 Funktionen

| Schritt | Beschreibung |
|---|---|
| **⬇️ Herunterladen** | Video von X/Twitter, YouTube oder direkter URL abrufen |
| **🎙️ Transkribieren** | Audio-Transkription via Groq (Whisper), 20+ Sprachen |
| **🌐 Übersetzen** | LLM-Übersetzung (DeepSeek / OpenAI) in die Zielsprache |
| **🎬 Einbrennen** | Untertitel-Overlay in das Video (ffmpeg/libass) |
| **☁️ Hochladen** | Lokaler oder S3-Speicher des Endergebnisses |

**Bonus:** VTT/SRT-Export, automatisches Wasserzeichen, Szenenerkennung, Inhaltsanalyse.

---

## 🎬 Pipeline-Ablauf

```
1. Video-Link einfügen
   │  YouTube, X/Twitter, TikTok, Instagram, Facebook...
   │  Validator prüft Zugänglichkeit in 2 Sekunden
   ▼
2. Herunterladen
   │  yt-dlp holt das Video (bis zu 4K)
   ▼
3. Audio-Transkription
   │  Groq Whisper wandelt Audio in Text um
   │  Unterstützt 20+ Sprachen
   ▼
4. Übersetzung
   │  DeepSeek / OpenAI übersetzt Untertitel
   │  In der Sprache Ihrer Wahl
   ▼
5. Einbrennen
   │  ffmpeg + libass brennen Untertitel in das Video ein
   │  Automatisches Wasserzeichen, anpassbarer Stil
   ▼
6. Ergebnis
   │  Untertitel-Video, bereit zum Teilen
   │  SRT/VTT-Export verfügbar
```

**Gesamtzeit:** 2-5 Minuten für ein 3-minütiges Video.
**Keine Programmierkenntnisse erforderlich.** Link einfügen, Sprache wählen, übersetztes Video erhalten.

## 🏗 Architektur

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  Frontend   │────▶│   Pipeline  │────▶│   Economy   │
│  Next.js    │◀────│  FastAPI    │◀────│  FastAPI     │
│  :3002      │     │  :8000      │     │  :8001       │
└─────────────┘     └──────┬──────┘     └─────────────┘
                           │                    │
                    ┌──────▼──────┐      ┌──────▼──────┐
                    │   Redis     │      │  PostgreSQL │
                    │  (queue)    │      │  (auth/bill)│
                    └──────┬──────┘      └─────────────┘
                           │
                    ┌──────▼──────┐
                    │   Celery    │
                    │  (worker)   │
                    └─────────────┘
```

**Pipeline** = reine Technik (dieses Repo). Ruft **Economy** per HTTP für die Authentifizierung auf.

## 🚀 Schnellstart

### Mit Docker (empfohlen)

```bash
# 1. Klonen
git clone https://github.com/Nansouoouu/subvox-pipeline.git
cd subvox-pipeline

# 2. Vollen Stack starten
export GROQ_API_KEY="your_groq_key"
export DEEPSEEK_API_KEY="your_deepseek_key"
docker compose up -d

# 3. Überprüfen
curl http://localhost:8000/health
```

### Ohne Docker (lokale Entwicklung)

```bash
# 1. Einrichtung
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 2. Konfiguration
cp .env.example .env
# → DATABASE_URL, REDIS_URL bearbeiten

# 3. Ausführen
uvicorn backend.main:app --reload --port 8000
```

## 📡 API

| Route | Methode | Beschreibung |
|---|---|---|
| `/jobs/feed` | GET | Neueste öffentliche Übersetzungen |
| `/jobs/{id}/status` | GET | Job-Status |
| `/jobs/{id}/subtitles` | GET | Generierte Untertitel |
| `/health` | GET | Health-Check |

Authentifizierung und Token-Verwaltung werden von einem dedizierten Dienst (privat) übernommen.

## 🧩 Stack

| Komponente | Technologie |
|---|---|
| API | Python 3.14 / FastAPI |
| Warteschlange | Celery + Redis |
| Video | FFmpeg 7 |
| Transkription | Groq (Whisper) |
> **Hinweis:** Groq bietet 2 Stunden kostenlose Transkription pro Tag. Perfekt für Entwicklung und kleine Projekte.
| Übersetzung | DeepSeek / OpenAI |
| DB | PostgreSQL 16 |
