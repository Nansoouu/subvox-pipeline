> これは一人のプロジェクトであり、コミットごとに積み上げられています。なぜなら、テクノロジーはすべての人のものであるべきだと信じているからです。英語を話す人だけのものではありません。

## 🎯 機能

| ステップ | 説明 |
|---|---|
| **⬇️ ダウンロード** | X/Twitter、YouTube、または直接URLから動画を取得 |
| **🎙️ 文字起こし** | Groq（Whisper）による音声文字起こし、20以上の言語に対応 |
| **🌐 翻訳** | LLM翻訳（DeepSeek / OpenAI）でターゲット言語へ |
| **🎬 字幕焼き込み** | ffmpeg/libassで動画に字幕を合成 |
| **☁️ アップロード** | 最終結果のローカルまたはS3ストレージ |

**ボーナス:** VTT/SRTエクスポート、自動ウォーターマーク、シーン検出、コンテンツ分析。

---

## 🎬 パイプラインの流れ

```
1. 動画リンクを貼り付け
   │  YouTube、X/Twitter、TikTok、Instagram、Facebook...
   │  バリデーターが2秒でアクセス可能性をチェック
   ▼
2. ダウンロード
   │  yt-dlpが動画を取得（最大4K）
   ▼
3. 音声文字起こし
   │  Groq Whisperが音声をテキストに変換
   │  20以上の言語に対応
   ▼
4. 翻訳
   │  DeepSeek / OpenAIが字幕を翻訳
   │  あなたが選んだ言語で
   ▼
5. 字幕焼き込み
   │  ffmpeg + libassが動画に字幕を焼き込み
   │  自動ウォーターマーク、カスタマイズ可能なスタイル
   ▼
6. 結果
   │  字幕付き動画、共有準備完了
   │  SRT/VTTエクスポート対応
```

**合計時間:** 3分の動画で2〜5分。
**コーディング不要。** リンクを貼り、言語を選び、翻訳済み動画を入手。

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  Frontend   │────▶│   Pipeline  │────▶│   Economy   │
│  Next.js    │◀────│  FastAPI    │◀────│  FastAPI     │

                           │                    │
                    ┌──────▼──────┐      ┌──────▼──────┐
                    │   Redis     │      │  PostgreSQL │
                    │  (queue)    │      │  (auth/bill)│
                    └──────┬──────┘      └─────────────┘
```

## 🚀 クイックスタート

### Dockerを使用する場合（推奨）

```bash
# 1. クローン
git clone https://github.com/Nansouoouu/subvox-pipeline.git
cd subvox-pipeline

# 2. フルスタックを起動
export GROQ_API_KEY="your_groq_key"
export DEEPSEEK_API_KEY="your_deepseek_key"
docker compose up -d

# 3. 確認
curl http://localhost:8000/health
```

### Dockerなし（ローカル開発）

```bash
# 1. セットアップ
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 2. 設定
cp .env.example .env
# → DATABASE_URL, REDIS_URL を編集

# 3. 実行
uvicorn backend.main:app --reload --port 8000
```

## 📡 API

| ルート | メソッド | 説明 |
|---|---|---|
| `/jobs/feed` | GET | 最新の公開翻訳 |
| `/jobs/{id}/status` | GET | ジョブのステータス |
| `/jobs/{id}/subtitles` | GET | 生成された字幕 |
| `/health` | GET | ヘルスチェック |

認証とトークン管理は専用サービス（非公開）によって処理されます。

## 🧩 スタック
